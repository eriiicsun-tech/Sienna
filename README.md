# Sienna Planner — Cloudflare Pages v2

This version is simplified and mobile-first.

## Changes
- Simpler layout
- No sample tasks preloaded
- Added calendar for important dates
- AI backend now defaults to gpt-4.1-mini
- Optional override model supported with OPENAI_MODEL

## Cloudflare environment variables
Add:
- OPENAI_API_KEY = your real OpenAI API key

Optional:
- OPENAI_MODEL = gpt-4.1-mini
